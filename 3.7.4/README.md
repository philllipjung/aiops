# Gradle Java Project

## 버전
**v3.7.4**

## 개요

Gradle 기반 Java 프로젝트입니다.

## 프로젝트 구조

```
3.7.4/
├── build.gradle.kts  # Gradle 설정 (Kotlin DSL)
├── gradle            # Gradle Wrapper
├── gradlew           # Gradle Wrapper (Unix)
└── build/            # 빌드 출력
```

## 빌드

```bash
cd /root/aiops/3.7.4

# 빌드
./gradlew build

# Clean
./gradlew clean
```
