# Maven Java Project

## 버전
**v9.4.5.3**

## 개요

Maven 기반 Java 프로젝트입니다.

## 프로젝트 구조

```
9.4.5.3/
├── pom.xml          # Maven 설정
├── mvnw             # Maven Wrapper
└── src/             # 소스 코드
```

## 빌드 및 실행

```bash
cd /root/aiops/9.4.5.3

# 빌드
./mvnw clean install

# 실행
java -jar target/*.jar
```
