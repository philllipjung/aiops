# Transport XHR Instrumentation

## 버전
**v3.4.2**

## 개요

Transport XHR Instrumentation v3.4.2는 XMLHttpRequest 호출 자동 계측 플러그인의 패치 버전입니다.

## 변경사항

- 버그 수정
- 호환성 개선

## 설치

```bash
npm install @grafana/transport-xhr-instr@3.4.2
```

## 사용

```typescript
import { XhrInstrumentation } from '@grafana/transport-xhr-instr';

const xhrInstr = new XhrInstrumentation();
```
