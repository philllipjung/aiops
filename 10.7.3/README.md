# AIOps Agent

## 버전
**v10.7.3**

## 개요

AIOps Agent v10.7.3는 OpenSearch와 통합된 로그/메트릭 수집 에이전트입니다.

## 기능

- Fluent Bit 기반 로그 수집
- OpenSearch 데이터 전송
- Docker 지원

## 설치

```bash
cd /root/aiops/10.7.3
docker-compose up -d
```
