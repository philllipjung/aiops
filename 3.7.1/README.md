# E2E Tests

## 버전
**v3.7.1**

## 개요

Faro Web SDK의 End-to-End 테스트 스위트 v3.7.1입니다. Cypress를 사용한 통합 테스트를 포함합니다.

## 테스트 항목

- SDK 초기화
- 이벤트 전송
- 에러 추적
- 세션 관리

## 실행 방법

```bash
cd /root/aiops/3.7.1
yarn install
yarn cypress
```
