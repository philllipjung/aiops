# Gradle Build Project

## 버전
**v4.4.4**

## 개요

Gradle 기반 Java 빌드 프로젝트입니다.

## 프로젝트 구조

```
4.4.4/
├── build.gradle        # Gradle 빌드 스크립트
├── gradle              # Gradle Wrapper
├── gradlew             # Gradle Wrapper (Unix)
└── gradlew.bat         # Gradle Wrapper (Windows)
```

## 빌드

```bash
cd /root/aiops/4.4.4

# 빌드
./gradlew build

# Clean
./gradlew clean
```
