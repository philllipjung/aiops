# ByteBuddy Java Agent

## 버전
**v3.8.1**

## 개요

ByteBuddy Java 바이트코드 조작 라이브러리 예제입니다.

## 주요 파일

```
3.8.1/
├── ByteBuddy/                    # 바이트코드 생성
├── ByteBuddyExampleMain.java    # 메인 예제
└── Robot.java                   # Robot 예제
```

## 실행

```bash
cd /root/aiops/3.8.1

# 컴파일
javac *.java

# 실행
java ByteBuddyExampleMain
```

## 주요 기능

- Java 바이트코드 생성
- 런타임 코드 조작
- Agent 개발
