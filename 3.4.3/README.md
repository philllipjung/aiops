# Python Microservices Demo

## 버전
**v3.4.3**

## 개요

OpenTelemetry을 사용한 Python 마이크로서비스 예제입니다. 서비스 간 통신과 추적을 보여줍니다.

## 프로젝트 구조

```
3.4.3/
├── __init__.py
├── service1.py      # 서비스 1
└── service2.py      # 서비스 2
```

## 실행

```bash
cd /root/aiops/3.4.3

# 서비스 1 실행
python service1.py

# 서비스 2 실행
python service2.py
```

## 주요 기능

- Python 마이크로서비스
- OpenTelemetry 추적
- 서비스 간 통신
