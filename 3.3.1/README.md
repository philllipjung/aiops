# Faro Component

## 버전
**v3.3.1**

## 개요

Faro Web SDK의 컴포넌트 패키지입니다. SDK의 모듈화된 부품을 포함합니다.

## 기능

- 모듈화된 컴포넌트 구조
- 재사용 가능한 빌딩 블록
- 타입 안전성

## 설치

```bash
cd /root/aiops/3.3.1
yarn install
```

## 사용

```typescript
import { Component } from '@grafano/component';
```
